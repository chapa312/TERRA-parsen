Инструкция пользователя: Парсер результатов TERRA 

Данная программа предназначена для автоматической обработки текстовых отчетов (.txt), созданных в программном комплексе термодинамических расчетов TERRA. Она самостоятельно извлекает данные об исходном составе и физико-химических параметрах, формируя удобную сводную таблицу в формате Excel (.xlsx).

Запуск программы

1. Поместите исполняемый файл parsing.v1.exe в любую удобную папку.

2. Запустите файл двойным кликом левой кнопки мыши.
Важно: При самом первом запуске программа может открываться 3-5 секунд — это нормальное явление, система распаковывает графический интерфейс.

Обработка результатов

1. Выбор папки с результатами
Нажмите кнопку «Обзор...» в самом верху окна и выберите папку, в которой лежат ваши текстовые отчеты из TERRA. (Заранее сохраняйте результаты в отдельную папку для удобства)

Сортировка: Программа сама отсортирует файлы по цифрам в их названиях (например: 1.txt, 2.txt, 10.txt), чтобы в итоговой таблице был строгий хронологический порядок.

2. Настройка исходного состава
Галочка «Исходный состав» включает выгрузку химических компонентов.

В меню «Кол-во компонентов» вы можете принудительно задать, сколько столбцов состава нужно выгрузить (от 1 до 7), либо оставить вариант «Все (Авто)». Во втором случае программа сама прочитает первый файл, определит химические формулы веществ и создаст нужное количество подписанных столбцов.

3. Выбор термодинамических параметров

По горизонтали находятся сечения сопла: Камера (кам), Критика (кр), Срез (вых).
По вертикали — список из 34 физических величин (Температура, Давление, Удельный импульс, Пок. адиабаты и т.д.).
Отметьте галочками только те данные, которые вам нужны для текущего анализа.

4. Сохранение результатов
Нажмите кнопку «Сформировать Excel-файл» внизу окна.
В появившемся окне выберите папку для сохранения и укажите имя для вашей новой таблицы (по умолчанию предложено Результаты_TERRA.xlsx).
Нажмите «Сохранить». Дождитесь всплывающего уведомления об успешном завершении работы!


Решение частых проблем
Ошибка: «Не удалось сохранить Excel файл»
Причина: Вы пытаетесь перезаписать таблицу, которая прямо сейчас открыта у вас в Microsoft Excel.
Решение: Закройте открытый Excel-файл и попробуйте нажать кнопку формирования заново.

Ошибка: «В выбранной папке нет .txt файлов»
Причина: Указан неверный путь к папке.
Решение: Убедитесь, что вы выбрали конечную папку, внутри которой физически лежат текстовые отчеты, а не ярлыки или вложенные папки.

В готовой таблице Excel есть пустые ячейки
Причина: Это не ошибка парсера. Если в отчете TERRA какой-либо параметр не рассчитывался (например, вместо значения числа Маха в камере стоит прочерк -), программа оставляет эту ячейку пустой, чтобы нули не искажали вам графики.
